# EMF Compat: Player Animator

## [Modrinth](https://modrinth.com/project/emf-compat-player-animator)

Makes Player Animator library animations work correctly with animated EMF player models.

Brings **Player Animator** full-body poses to **Entity Model Features**.

When a resource pack replaces the player model with an animated Entity Model Features variant, animations driven by the Player Animator library (used by Better Combat and others) are often overwritten. This addon captures the full body pose while the animation is active, then restores it after EMF runs so modded player animations stay visible.

Tested with **Fresh Animations: Player Extension**.
