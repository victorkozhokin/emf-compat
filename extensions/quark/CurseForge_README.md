# EMF Compat: Quark

![Headbang!](https://cdn.modrinth.com/data/UvuFGp0l/images/444839c0908ae3322a66ab7a8b28eec190659c2b.gif)

A small client-side mod that makes **[Quark](https://www.curseforge.com/minecraft/mc-mods/quark)** emotes work correctly with **[Entity Model Features](https://www.curseforge.com/minecraft/mc-mods/entity-model-features)** player models.

Tested with **[Fresh Animations: Player Extension](https://www.curseforge.com/minecraft/texture-packs/fa-player-extension)** but it should work with any player animation resource pack.

## Features

- Compatible with **[Fresh Animations: Player Extension](https://www.curseforge.com/minecraft/texture-packs/fa-player-extension)**.
- Wave, salute, dance and other Quark emotes play correctly over EMF animations.
- Only the arms and body parts used by the emote change; the rest keeps its resource-pack animation.
- Should work with most player animation resource packs using EMF.
